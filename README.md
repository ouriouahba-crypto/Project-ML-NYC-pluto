# Project ML - NYC PLUTO

Student machine learning project applying clustering techniques to the NYC PLUTO dataset, developed by a team of three second-year Data Engineering students at Bar-Ilan University.

## Data

### Raw Data (`data/raw/`)
- `projetML.csv.gz` — **Original, unmodified dataset.** This is the source of truth. Do not edit files in this folder.

### Processed Data (`data/processed/`) *(to be added)*
- Modified versions of the dataset will be stored here as the project progresses.
